# Oral_Health_Project
This repository contains files (code, graphs, presentation, manuscript, etc.) related to the research project "Development and Validation of the US Adults Oral Health Prediction Model" under authors Zhixin Cao, Yuhao Wang and David Kim.

## Oral_Health_Powerpoint
This powerpoint presentation is an overall summary of the project with detailed description for background, literature review, data collection, methodology and results. It also demonstrates the outline of the written manuscript, though certain changes in the "Methodology and Results" part have been made. Please refer to the manuscript for final version of the project.
